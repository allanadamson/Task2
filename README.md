Projekti pealkiri: Store Inventory Management System
Kirjeldus

See on Tallinna Tehnikaülikooli õppetöö raames valminud veebirakendus tooteandmete haldamiseks. Rakendus võimaldab vaadata tootenimekirja, otsida tooteid ning lisada uusi tooteid, kasutades TypeScripti ja LocalStorage'it andmete säilitamiseks.

Kasutatud tehnoloogiad

TypeScript – tüübikindel arendus.

HTML5 & CSS3 – kasutajaliides ja kujundus.

Bootstrap – tundlik disain (Responsive Design).

LocalStorage – andmete püsivus brauseris ilma eraldi andmebaasita.
